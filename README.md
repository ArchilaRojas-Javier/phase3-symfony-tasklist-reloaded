# Tasklist-Reloaded

Une application de gestion de tâches priorisées et organisées en dossiers. Développée avec Symfony.

## Dependencies

-PHP 8.5 
-Symfony CLI 8.0
-Composer

## Installation

```bash
git clone https://github.com/ArchilaRojas-Javier/phase3-symfony-tasklist-reloaded.git

composer require install


```
# Running dev

```bash
symfony server:start
```
